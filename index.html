学记助理开发日志:项目启动

发表于 2019-12-01 | 分类于 项目

学记助理 是一款智能知识管理应用。目的是探索一种新的知识载体与交互方式，实现知识网络、团队协作、学习与知识共享的一体化，化繁为简，降低信息熵，为机器建立常识，帮助人们积累知识财富。目标是成为通用应用、知识银行、第二大脑，让每个人可以自由学习与创新！

标签

智能知识管理、对话式人机交互、通用文档画板、语义计算、知识图谱、智能助理、人类计算（众包合作）、贡献积分代币。

概述

产品：基于统一语义，结合智能助理的智能文档，通用应用（首先用于知识管理）
愿景：每个人的智能学习助理，自由高效学习与创新
技术：增强学习、语义计算、知识图谱，语义web+app
运营：精准核心内容、工具模板、贡献积分代币，共享经济

项目简介

协同知识管理应用概念原型及项目计划

  • 核心业务

智能知识管理：收集（获取知识）、整理（理解）、存储、应用（检索、规划管理、学习、共享、教学）、创新（协作、推理）。非功能性：安全、可靠、准确、可信、权限、实时。

  • 解决方案

基于统一语义，结合智能助理的对话式交互多视图画板、智能模板，特色：简捷高效、精准、可视化、智能交互、安全、贡献积分、代币激励分配。

  • 核心模型

语义计算、文档模型、知识图谱、认知智能（用户+助理）、空间系统（事物）。

  • 第一版功能

协同构建知识图谱（核心语义、个人知识、开放知识）+多视图（纯文本卡片导图）的文档画板+3D非智能助理。

  • 里程碑

2019年12月31日第一版上线。基本功能：多用户多视图文档画板，实现内容管理+知识图谱协作功能。初步领域模型、基础组件、语义计算+智能引擎+3D助理+DSL。

  • 任务

UI原型设计+数据模型设计+技术架构+开发环境（领域模型+测试驱动）、网站：版本历史及路线图。

Hexo博客搭建

发表于 2019-04-10

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

准备

安装 Nodejs + yarn + Hexo + Git + GitHub 账号

编辑生成网站

配置

  • title: Aaron ZZH - 专注智能知识管理
  • subtitle: 大道至简 以人为本
  • description: 个人博客，C++/js全栈工程师，语义 Web、图数据库开发，知识管理、人工智能爱好者
  • url: https://zzhgit.github.io
  • language: zh-CN //在主题 languages 目录下有对应文件

注意：配置项冒号后空格

部署

安装 hexo-deployer-git

创建仓库

ZZHGit.github.io

deploy:
type: git
repo: // 如：https://github.com/ZZHGit/ZZHGit.github.io.git
branch: master

绑定域名

新增域名解析规则

新增 CNAME，记录值为：[username].github.io

新增 CNAME 文件

在项目的 .\source\ 路径下，新增 CNAME 文件，输入你的域名，如：zzh.xuejiai.com。
重新新生成文件，预览无误后，推送到GitHub
